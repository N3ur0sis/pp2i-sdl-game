\doxysubsection{Detailed Description}
Represent a single texture in the engine. Can be used by multiple Objet, completely dissociated from any Model, this class is just the reuslt of loading an image into a buffer. 

\begin{DoxyNote}{Note}
It uses the default texture unit, hence it is not necessary to specify the uniform value on the shader. 
\end{DoxyNote}
