void ChangeSceneEvent ( int  nextSceneIndex)

Creates and pushes a scene change event.

This function creates a custom SDL event for changing scenes and pushes it to the SDL event queue. It allocates memory for the next scene index and assigns it to the event's user data.

nextSceneIndexThe index of the next scene to switch to.

void HandleChangeScene ( SceneManager *  sceneManager)

Handles changing the current scene in the scene manager.

This function processes the scene change event and updates the scene manager accordingly. It unloads the current scene and sets the new scene based on the next scene index stored in the game state.
